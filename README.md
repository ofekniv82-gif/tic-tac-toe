# Assignment 1 - Tic Tac Toe

**Authors:** Ofek Saar, Niv Shapira

In this assignment, you need to develop a Tic Tac Toe game.

## Instructions
1. The game is for two players that play on one android device.
2. The first player is the “X” and the second is the “O”.
3. When the game ends an appropriate message will be presented on the screen and a “play again” button will be available.
